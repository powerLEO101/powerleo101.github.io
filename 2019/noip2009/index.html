Hankson 的趣味题

首先因为 x 是 b1 的因数，所以先枚举 x，也就是枚举 b1 的因数

接下来就是判断合不合法了

有两种判断方法

方法 1：

看 x 是否满足 $x * b0 == b1 * gcd(x1, b0)$ 和 $x *a0 = a1 * lcm(x1, a0)$

这个很好解释，原理是 $lcm(a, b) * gcd(a, b) == a *b$，然后根据这个式子退出 x 要满足的式子，化简一下就可以了

方法 2：

看 x 是否满足 $gcd(b1 / x, b1 / b0) == 1$ 和 $gcd(a0 / a1, x / a1) == 1$

gcd 和 lcm 的性质

显然方法 2 比方法 1 更优，因为 lcm 复杂度更大

最优贸易

建一个分层图，有 3 层

第一层到第二层的边表示买入，边权为负

第二层到第三层的边表示卖出，边权为正

然后跑一遍 spfa 就可以了
