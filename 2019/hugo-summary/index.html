hugo搭建博客使用总结

Written by Leo101 with ♥ on April 13, 2019 in 各种总结

在使用本文章之前，你需要有一定的 git 基础

安装

hugo 的特点是完全没有依赖，所以安装特别简单

sudo apt install hugo

使用

新建网站

通过hugo new site ***来新建一个网站

hugo 会在你的当前目录新建一个文件夹，表示网站

选择主题

hugo 默认没有主题所以要自己下

之前 worddpress，typecho 那些花里胡哨的主题看多了觉得没意思，主要是麻烦，再就是没钱

不得不说 hexo 的博文数量和社区活跃度还是远超 hugo，但是因为 hugo 优秀的依赖关系（没有），我还是用了 hugo

你用了 hexo 就会发现 npm 跟 shi 一样

这里选择了 leaveit，比较简单，但也好看，也方便日后 diy

https://themes.gohugo.io/leaveit/

将主题 clone 到 theme 文件夹，然后在 config.toml 追加

theme = "LeaveIt"

主题配置

把主题文件夹里的 exampleSite 里的 config 蒯过来就差不多了

根据代码你会发现会有一个 About 是 404，只需要创建 about.md 这篇文章就会自动读取

其他的就是一点小修改，不列举了

写文章

使用命令

hugo new ***

文章文件会生成在 content 里，直接写就可以了

然后根据 LeaveIt 的逻辑，blog 需要存在 posts 里

所以在 LeaveIt 下新建 blog 的文章的命令是：

hugo new posts/***

当然你也可以通过修改 config.toml 来更改 blog 的储存目录

其中同步配置文件里的 draft 如果为 false 才会生成到网站里

注：about.md 不需要存在 posts 里

在线预览

hugo server -D

在http://localhost:1313访问网站

生成静态网页

直接使用命令

hugo

即可

网页会生成到 public 文件夹里，直接将这个文件夹部署到 github 上即可
